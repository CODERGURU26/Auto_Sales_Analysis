# Excel Sales Dashboard

## Project Overview
This project analyzes sales data using Microsoft Excel. Pivot Tables and charts were used to build an interactive dashboard showcasing key business metrics.

## Key KPIs
- Total Sales
- Total Orders
- Average Order Value

## Visualizations
- Monthly Sales Trend (Line Chart)
- Top 5 Product Lines by Sales (Bar Chart)
- Deal Size Contribution (Pie Chart)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Formulas
- Data Cleaning & Analysis

## Key Insights
- Classic Cars generated the highest revenue
- Medium deal size contributed the largest share of sales
- Sales peaked during Q4 months
